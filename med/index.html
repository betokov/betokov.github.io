Медсправка водительская 2020 по новой форме в Москве

Ищите, где пройти медкомиссию для получения справки на водительские права в Москве? Пройти водительскую медкомиссию и оформить справку с психиатром и наркологом можно в одном из 17 наших медицинских центров! Записаться на медкомиссию для получения или замены водительских прав можно по телефону или через форму... далееонлайн заявки.

Водительская медкомиссия проводится в одном месте без очереди! Услуги по проведению медицинских осмотров проводят квалифицированные врачи. По результату вы получите медицинский документ нового образца 2020 года по форме 003-в/у. Справка для ГИБДД позволит получить новые права или заменить водительское удостоверение по истечении срока действия за 1 день. Цена от 1200 рублей. Уточняйте стоимость по телефону, цены могут отличаться в зависимости от выбранного филиала прохождения медкомиссии на водительские права.

Медсправка водительская по новой форме 003-в/у

Для получения водительского удостоверения на право управления транспортными средствами необходимо в ГИБДД предоставить «Медицинское заключение о наличии (об отсутствии) у водителей транспортных средств (кандидатов в водители транспортных средств) медицинских противопоказаний, медицинских показаний или медицинских ограничений к управлению транспортными средствами» по новой форме 003- В/у.

  • Справка для категорий А, А1, В, В1, М, ВЕ
  • Справка для категорий С, С1, D, D1, CE, C1E, D1E, DE, Tm, Tb
  • Справка для международных водительских прав
  • Справка для погрузчиков и тракторов
  • ЭЭГ (электроэнцефалограмма головного мозга) в ГИБДД
  • Справка для профессиональных водителей для работы
